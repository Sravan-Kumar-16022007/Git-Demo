## Files in this repository

- **calculator.py** - A basic calculator with add, subtract, multiply, and divide operations
- **Hello.py** - Python hello world file
- **Welcome.java** - Java welcome file

## Running the Calculator

To run the calculator, use:
```bash
python calculator.py
```

The calculator provides an interactive menu with the following operations:
1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Exit
